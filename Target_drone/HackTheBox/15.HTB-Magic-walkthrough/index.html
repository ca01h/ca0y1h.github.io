运行

sysinfo显示出来的内容主要分为四个部分

  • Hardware Info
  • Disk Info
  • CPU Info
  • Mem Usage

查阅了Forum上前辈们给的提示，发现sysinfo是四个Linux命令的组合调用：

  • Hardware Info = lshw -short
  • Disk Info = fdisk -l
  • CPU Info = cat /proc/cpuinfo
  • Mem Usage = free -h

既然sysinfo调用的是原生的Linux命令，那么我们就可以通过改变PATH变量，创建一个内容由自己决定，文件名为上面四种任意一个的“二进制文件”，具体而言：
